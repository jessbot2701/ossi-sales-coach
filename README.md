# Ossi – AI Sales Coach

Aktuelle Website-Version: `1.0`

Eine responsive Landingpage für einen ElevenLabs Voice Agent, der realistische B2B-Kaltakquise im SAP-Umfeld trainiert.

## Funktionen

- ElevenLabs Conversational AI Widget
- dreistufiger How-to-Guide
- kopierbare Gesprächseinstiege
- responsives, barrierearmes Design
- offizielle IGZ-Primärfarbe `#004482`
- großes Ossi-Portrait als Hero-Bild
- optimierte WebP-Bildauslieferung mit PNG-Fallback
- versioniertes Voice-Widget wird erst nach Zustimmung geladen

## Lokal starten

```bash
python3 -m http.server 4173
```

Danach `http://localhost:4173` öffnen.

## Varianten

- `index.html`: Hauptversion mit surrealer Ossi-Karikatur
- `v2.html`: Referenzkopie der surrealen Karikatur-Version

## Datenschutz

Die Seite enthält einen Hinweis zur Sprachverarbeitung und lädt das ElevenLabs Voice Widget erst nach aktiver Zustimmung. Für einen produktiven Einsatz müssen die konkreten Datenschutz- und Betriebsanforderungen der verantwortlichen Organisation geprüft werden.
